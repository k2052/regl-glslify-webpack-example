# regl-glslify-webpack-example

A simple [regl](https://github.com/regl-project/regl) example using webpack and glslify

[Demo](https://pawsong.github.io/regl-glslify-webpack-example/)

## Install

```bash
$ npm install
```

## Run

```bash
$ npm start
```

## Build

```bash
$ npm run build
```
